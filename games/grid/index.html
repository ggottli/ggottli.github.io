How to Play

The Grid challenges you to fill a 5×5 board with cards from a shuffled deck.

  1. The first card is placed in the center. Choose empty cells that touch an existing card.
  2. If only one neighbor touches your chosen cell, guess whether the new card will be Higher or Lower.
  3. If two or more neighbors touch it, guess whether the new card lies Inside or Outside the neighbor range.
  4. Correct guesses place the card. Wrong guesses clear that cell's entire row and column, then the remaining cards collapse toward the top-left.
  5. Fill all 25 cells to win. Run out of cards first and you lose.

Ties always count as wrong, so choose wisely!

Keyboard Shortcuts
  • Arrow Keys - Navigate the grid
  • Space - Select focused cell
  • 1-4 - Make guesses when a cell is selected
  • N - New game
  • R - Show/hide rules
  • Escape - Close dialogs
